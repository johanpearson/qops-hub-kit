Azure Functions API demonstrating file upload with multipart/form-data using @qops/hub-kit

üìã Overview

This API demonstrates how to implement file upload functionality in Azure Functions using the @qops/hub-kit library. It supports multipart/form-data requests with file uploads and form field validation using Zod schemas.

üöÄ Endpoints

POST /api/upload Multipart/Form-Data
Upload file with metadata
Upload one or more files along with metadata using multipart/form-data

Request Parameters

Field Type Required Description
title string ‚úì Required Title of the upload (minimum 1 character)
description string Optional Optional description of the upload
category enum Optional Category: document, image, or other
file binary ‚úì Required File(s) to upload (supports multiple files)

üí° Example Usage

Using curl

curl -X POST http://localhost:7071/api/upload \
  -F "title=My Document" \
  -F "description=A test document" \
  -F "category=document" \
  -F "file=@/path/to/your/file.txt"

Multiple Files

curl -X POST http://localhost:7071/api/upload \
  -F "title=Multiple Files" \
  -F "file=@/path/to/file1.txt" \
  -F "file=@/path/to/file2.pdf"
